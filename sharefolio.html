Sharefolio Analyser ASX

An award-winning ASX investment portfolio management tool.

Try Sharefolio Analyser ASX

Track Your Investments Like a Pro

Launched in July 2022 and previously known as the ASX Dashboard, Sharefolio Analyser ASX is a portfolio tracker for Australian Securities Exchange investors. Users can upload their trade and dividend history in CSV or manually enter their records via the platform. Sharefolio Analyser ASX then analyses their portfolio performance and provides insights with interactive charts.

Sharefolio Analyser ASX is a full-stack web application built with Python, SQL and HTML. By integrating finance and accounting knowledge, this project is a result of an interdisciplinary learning journey and a passion for building technological tools that empower human beings.

Sharefolio Analyser ASX was awarded the Best Junior Undergraduate Project by Citadel Securities at the Coding Fest 2023 at the University of Sydney.



Tech Stack

  • Backend: Python (Flask), SQL Lite
  • Data Analysis: Python (Pandas, Numpy)
  • Front End: HTML, CSS, Javascript
